1. LICENCIAS

Artefacto: Licencias

La Licencia de cualquier tipo de su uso en un programa de software, es en si un documento que de forma legal, define los terminos y condiciones para usar el software de forma legal. Este tipo de artefacto es esencial para tener regulación entre el titulas de derechos de autor y proveedor de software con el usuario final.

Caracteristicas principales

1. Identificacion del software: Nombre del Software a vender, incluyendo la version que se refiere.

2. Derechos principales: Derechos que se van a otorgar al usuario que haga la compra de este.

3. Duración de Licencia: Fechas de duración en el uso de la licencia.

4. Restricciones: Enumeración de acciones prohibidas, modificación, redistribución.

3. DISPOSITIVOS

Artefacto: Dispositivos TI

Los Dispositivos TI, ya sea como alguna Laptop, Celular, Impresora o demás, son herramientas fisicas que estan diseñadas para realizar funciones especificas con el procesamiento, almacenamiento, de la impresion y visualizacion de documentos; estos estan realizados para facilitar las tareas y mejorar la productividad en entornos profesionales.

Caracteristicas principales

1. Hardware: Componentes fisicos como memoria, procesador, pantalla, etc.

2. Software: Programa o sistema operativo con los que ya cuenta el dispositivo.

3. Conectividad: Capacidad de conexión para Wifi, Blueetooth, y demás.

4. Diseño: Diseño tal cual del dispositivo, para la mayor comidad del usuario, asi como tamaño, color, peso, demás.

5. Mantenimiento: Contar con la probabilidad de realizar mantenimiento constante para la solucion de problemas y rapidez del dispositivo.
